  <name>Task 1: Create auth module (GHL Ed25519) and rate limiter module</name>
  <files>agent/auth.py, agent/limiter.py, .env.example</files>
  <action>
1. Create `agent/auth.py` with GHL Ed25519 signature verification:
   - Hardcode the GHL public key PEM (this IS a public key, not a secret):
     ```
     -----BEGIN PUBLIC KEY-----
     MCowBQYDK2VwAyEAi2HR1srL4o18O8BRa7gVJY7G7bupbN3H9AwJrHCDiOg=
     -----END PUBLIC KEY-----
     ```
   - Load key at module level: `serialization.load_pem_public_key(GHL_PUBLIC_KEY_PEM)` cast to `Ed25519PublicKey`
   - Function `verificar_firma_ghl(raw_body: bytes, signature_b64: str) -> bool`:
     - `base64.b64decode(signature_b64)` to get raw signature bytes
     - Call `_ghl_pubkey.verify(sig, raw_body)` — returns None on success, raises InvalidSignature on failure
     - Return True if valid, False on InvalidSignature or any other Exception
   - Import from `cryptography.hazmat.primitives.asymmetric.ed25519` (already a transitive dep of anthropic — no new install needed)
   - Add logger for rejected signatures

2. Create `agent/limiter.py` with per-phone rate limiting:
   - `import os` and `from cachetools import TTLCache` (cachetools already installed by Plan 01)
   - `_RATE_LIMIT = int(os.getenv("RATE_LIMIT_PER_MINUTE", "15"))` — default 15/min per research recommendation
   - `_counters: TTLCache = TTLCache(maxsize=10_000, ttl=60)` — 60 second window
   - Function `verificar_rate_limit(telefono: str) -> bool`:
     - Get current count from `_counters.get(telefono, 0)`
     - If count >= `_RATE_LIMIT`: return False (over limit)
     - Otherwise: increment `_counters[telefono] = count + 1` and return True
   - Add `RATE_LIMIT_MESSAGE` constant: `"Estas enviando muchos mensajes seguidos. Por favor espera un momento antes de continuar."`

3. Update `.env.example` — add these new variables (with comments):
   ```

   WHAPI_WEBHOOK_SECRET=
   # Set to true to reject GHL webhooks without valid X-GHL-Signature
   GHL_WEBHOOK_AUTH_STRICT=false
   # -- Rate Limiting --
   # Max messages per phone per minute before rate limiting kicks in
   RATE_LIMIT_PER_MINUTE=15
   ```

<task type="auto">
  <name>Task 2: Wire auth and rate limiting into main.py webhook handlers</name>
  <files>agent/main.py, tools/configure_whapi_webhook.py</files>
  <action>
1. **agent/main.py — Whapi webhook auth** (TECH-04):
   At the TOP of `webhook_handler` (POST /webhook), before parsing the body:
   ```python
   WHAPI_WEBHOOK_SECRET = os.getenv("WHAPI_WEBHOOK_SECRET", "")
   # ... in handler:
   if WHAPI_WEBHOOK_SECRET:
       token = request.headers.get("X-Whapi-Token", "")
       if token != WHAPI_WEBHOOK_SECRET:
           logger.warning(f"Webhook Whapi rechazado - token invalido")
           raise HTTPException(status_code=401, detail="Unauthorized")
   ```
   Load the env var at module level (not inside the handler) for performance.

2. **agent/main.py — GHL webhook auth** (TECH-05):
   In `ghl_webhook_handler` (POST /webhook/ghl):
   - CRITICAL: Read `raw_body = await request.body()` FIRST, before any JSON parsing
   - Check for `X-GHL-Signature` header
   - If header present: call `verificar_firma_ghl(raw_body, sig)` — reject with 401 if invalid
   - If header absent AND `GHL_WEBHOOK_AUTH_STRICT` is "true": reject with 401
   - If header absent AND strict is false (default): allow through with a warning log
   - Parse body with `body = json.loads(raw_body)` instead of `await request.json()` (avoid double-read pitfall)
   - Add `from agent.auth import verificar_firma_ghl` import at top

3. **agent/main.py — Rate limiting** (TECH-06):
   Inside `webhook_handler`, in the message loop, AFTER the dedup check (es_duplicado) and BEFORE the historial/generar_respuesta calls:
   ```python
   from agent.limiter import verificar_rate_limit, RATE_LIMIT_MESSAGE
   # ... inside loop:
   telefono_normalizado = normalizar_telefono(msg.telefono)  # already done by Plan 01
   if not verificar_rate_limit(telefono_normalizado):
       logger.warning(f"Rate limit excedido para {telefono_normalizado}")
       await proveedor.enviar_mensaje(msg.telefono, RATE_LIMIT_MESSAGE)
       continue  # Skip Claude API call
   ```
   Note: Use normalized phone for rate limit key (consistent counting regardless of format). Use original msg.telefono for sending (Whapi needs @s.whatsapp.net format).
