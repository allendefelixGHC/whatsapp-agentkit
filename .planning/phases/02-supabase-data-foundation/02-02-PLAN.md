2. **Modify `lifespan` function**: After `await inicializar_db()`, add:
   ```python
   try:
       await cargar_cache_desde_supabase()
       logger.info("Cache de propiedades cargado desde Supabase")
   except Exception as e:
       logger.warning(f"No se pudo cargar cache de propiedades desde Supabase: {e}")
       logger.warning("El bot funcionara sin cache de propiedades hasta el primer refresh")
   ```
   The try/except is important: if SUPABASE_URL/KEY are not configured yet, the server should still start (graceful degradation).

3. **Add admin token** env var reading near the top (after GHL_WEBHOOK_AUTH_STRICT):
   ```python
   ADMIN_TOKEN = os.getenv("ADMIN_TOKEN", "")
   ```

4. **Add `/admin/refresh-properties` POST endpoint**:
   ```python
   @app.post("/admin/refresh-properties")
   async def admin_refresh_properties(request: Request):
       """Triggers full scrape of Bertero + upsert to Supabase + cache reload.
       Called by n8n Schedule Trigger every hour."""
       # Auth: require X-Admin-Token header
       if ADMIN_TOKEN:
           token = request.headers.get("X-Admin-Token", "")
           if token != ADMIN_TOKEN:
               raise HTTPException(status_code=401, detail="Unauthorized")

       try:
           # 1. Scrape + persist to Supabase
           stats = await scrape_and_persist()

           # 2. Reload in-memory cache from Supabase
           await cargar_cache_desde_supabase()

           logger.info(f"Refresh completado: {stats}")
           return {"status": "ok", "stats": stats}
       except Exception as e:
           logger.error(f"Error en refresh de propiedades: {e}")
           raise HTTPException(status_code=500, detail=str(e))
   ```

5. **Add ADMIN_TOKEN to .env** (append):
   ```
   # -- Admin --
   ADMIN_TOKEN=
   ```
   If empty, auth is skipped on admin endpoints (dev mode). In production, set a strong random token.

IMPORTANT: Do NOT modify any existing endpoints (webhook, webhook/ghl, health_check). Only ADD new code.
